How it works

A growing collection
of features.

Step 01

Write to SMC

Batty writes directly to the System Management Controller using Apple-private CH0C and ACLC keys. This is the same mechanism macOS Optimized Charging uses internally.

Step 02

Hardware stops charging

Once the key is set the battery controller enforces the limit in firmware. The Mac stops accepting charge at exactly your target percentage, even while sleeping.

Step 03

Persists across reboots

A single sudoers entry lets the app write SMC keys without a daemon. No background process, no scheduled tasks. The limit is retained by the hardware itself.

No software polling. SMC keys are set at the firmware level so charging stops at your exact target, even while the lid is closed.

No code signing required

Run ad-hoc or unsigned. No Apple Developer account, no notarization, no entitlements. Just grant sudo permission once.

Under the hood

Writes directly
to SMC keys.

Apple Silicon exposes charging control through the System Management Controller. Batty uses SMCKit to write specific keys that instruct the battery controller to hold at your chosen limit.

The limit persists in hardware — no polling loop required. When you change it, Batty writes the new value and exits. The firmware does the rest.

  • Uses the same mechanism as macOS Optimized Charging
  • sudoers rule grants smc write without a full-daemon
  • Battery data read via IOKit AppleSmartBattery
  • Adapter detection via AppleSmartBatteryManager
SMC Key Purpose
CH0C Disable / enable hardware charging
ACLC Set charge limit percentage
BCLM Battery charge level maximum
B0FC Full-charge capacity (mAh)
B0RM Remaining capacity (mAh)
TB0T Battery temperature
Get started

Up and running
in two minutes.

Download the DMG

Recommended for most users. No Terminal required.

  1. 1
    Download Batty.dmg from the Releases page
  2. 2
    Open the DMG and drag Batty to /Applications
  3. 3
    Launch Batty. It appears in the menu bar and shows a one-time setup screen
  4. 4
    Click Grant Permission and enter your admin password once
  5. 5
    Open the menu bar icon → Settings → set your limit → tap Apply
Download Batty.dmg

Build from source

For developers. Requires Xcode Command Line Tools only.

git clone https://github.com/aliadelelroby/Batty.git
cd Batty
bash install.sh

The installer builds a release binary, creates /Applications/Batty.app, and handles the sudoers setup in one step.

Uninstall 

sudo rm /etc/sudoers.d/batty
sudo rm -rf /Applications/Batty.app
Open source

Built in the open.
Contributions welcome.

Batty is MIT-licensed, built with Swift Package Manager (no Xcode project), and has zero external dependencies. The codebase is small and approachable.

The focus is keeping it simple, fast, and reliable. If you want to add something, open an issue first to discuss. We'd rather have one excellent feature than five mediocre ones.

View on GitHub Open an issue
  • Bug fixes Always welcome. Include macOS version, Mac model, and relevant log show output.
  • Intel Mac support Intel uses different SMC keys (CH0C for charging, CH0J for adapter).
  • Battery detail rows Additional IOKit properties worth surfacing in the Details tab.
  • Time remaining accuracy Improvements to the coulomb-counter based time estimation.
